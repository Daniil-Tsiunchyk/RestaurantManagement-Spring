# RestoHRM - Система управления персоналом для ресторанного бизнеса

RestoHRM - это новаторское программное решение, разработанное специально для HR-менеджмента в ресторанном бизнесе. Система, созданная на Java и фреймворке Spring Boot, интегрирована с базой данных MySQL, что обеспечивает высокую производительность и безопасность данных.

## Основные функции

RestoHRM предлагает множество возможностей:

- Управление информацией о сотрудниках: ведение данных о всех сотрудниках, включая официантов, поваров, менеджеров и администраторов.
- Планирование рабочих смен: эффективное планирование рабочих смен и выходных.
- Мониторинг выполнения задач и KPI: отслеживание выполнения задач и индивидуальных KPI.
- Журнал заказов: управление заказами и координация деятельности между официантами и кухней.
- Отслеживание ингредиентов и продуктов: отслеживание ингредиентов, продуктов и мониторинг их использования.
- Управление доступом: контроль доступа к системе с учетом ролей и полномочий.

## Используемые технологии и инструменты

- Язык программирования: Java
- Фреймворк: Spring Boot
- База данных: MySQL
- ORM: Hibernate
- Система контроля версий: Git

Для запуска приложения убедитесь, что на вашей системе установлены Java и MySQL. Затем следуйте этим шагам:

1. Склонируйте репозиторий с GitHub.
2. Не забудьте настроить файл application.properties, указав свои учетные данные MySQL.
3. Используйте Maven для сборки проекта, выполнив команду `mvn clean install`.
